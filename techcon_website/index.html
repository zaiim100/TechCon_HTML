Welcome to My First Webpage

This is the very first webpage I am creating as part of the ALX Software Engineering program.

About This Project

The goal of this project is to learn the basics of HTML: how to structure a webpage, use tags, and create simple elements like headings, paragraphs, and links.

What You Will Find

In this webpage, you will see examples of HTML tags including headings, paragraphs, and an image with a clickable link.

As I continue learning, I will add more features and build complete websites using HTML, CSS, and JavaScript.
